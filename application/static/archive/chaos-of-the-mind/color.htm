The Color Order

Thispage should be used in conjunctoin with the Applying Wax page, in the main proccess. Basicly it is the order in which you should dye the egg to avoid creating an ugly brownish black color when trying to add another color.

1. White
2. Tuorquise *
3. Yellow
4. Green *
5. Orange
6. Red (or Pink)
7. Black (or Brown)

* These advanced colors should be added with a toothpick (dip the toothpick in the dye, and apply it directy to the egg.) Then these colors should be covered with beeswax before the regular proccess is resumed. They should only be used by those with at least some expirence creating eggs.
